# Zepto-Ecommerce-SQL-Analysis
PostgreSQL-based e-commerce inventory analysis project featuring data cleaning, exploratory analysis, and business intelligence reporting.

This project demonstrates an end-to-end SQL Data Analytics workflow using a real-world e-commerce inventory dataset from Zepto. The analysis covers database creation, data cleaning, exploratory data analysis (EDA), and business-focused SQL queries to generate actionable insights. The project showcases practical SQL skills used by data analysts, including aggregations, filtering, joins, CTEs, window functions, and business reporting.
## Project Overview
This project analyzes Zepto's inventory dataset using PostgreSQL.
